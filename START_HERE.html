Source Card Schema

A local-only starter repo for turning links, documents, claims, and signals into durable source cards.

Boundary: this tool does not fetch sources, verify claims, clear rights, provide advice, or publish anything. Treat source locators as text.

Start

python3 scripts/validate_source_card.py examples/source-card.example.json
python3 scripts/render_source_card_html.py examples/source-card.example.json /tmp/source-card-example.html

Use

Copy one of the synthetic examples, replace the field values with your own public-safe notes, validate the card, then render a local HTML review page.

Public Safety

Do not include private source relationships, credentials, account data, unpublished or private materials, sensitive traces, client data, or internal machinery in cards intended for public release.
