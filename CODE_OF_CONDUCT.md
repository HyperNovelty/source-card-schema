# Code of Conduct

This project is intended for practical public-good source review work. Participants are expected to communicate with care, keep examples synthetic, respect privacy boundaries, and avoid exposing sensitive information.

## Expected Conduct

- Be clear, factual, and constructive.
- Keep criticism focused on the work.
- Respect requests to remove private or sensitive details.
- Avoid adding credentials, account data, private source relationships, unpublished materials, client details, or sensitive traces.

## Unacceptable Conduct

- Harassment or targeted abuse.
- Publishing private information without consent.
- Using the project to expose protected source relationships.
- Presenting source cards as legal, compliance, medical, financial, or rights-clearance authority.

Maintainers may remove contributions that violate these boundaries.
